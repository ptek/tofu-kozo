# tofu-kozo :: 豆腐小僧
## An HTTP-based control scheme for [phantom.js](http://www.phantomjs.org/phantom.js)


#### Aim

To dynamically control phantom.js through a language-agnostic HTTP interface.

#### Requirements

 * Phantom.js 1.5.0

#### Development requirements
 * curl
 * Ruby + Bundler
 * Rspec 2.9
